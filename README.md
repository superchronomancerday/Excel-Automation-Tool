## ❗ Troubleshooting

| ❌ Problem | ✅ Fix |
|---|---|
| Macros blocked | Enable macros in Excel Trust Center settings |
| Batch processor skips files | Ensure workbooks are not open in another Excel instance |
| Pivot table empty | Check source range, ensure headers are present |
| Dashboard slicers broken | Reconnect slicer to pivot table in Dashboard Creator |
| Formula links broken | Run Formula Linker audit to detect and repair references |
| Performance still slow | Convert volatile formulas, reduce conditional formatting ranges |
